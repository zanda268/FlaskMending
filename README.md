## Ruined Mending

A spiritual successor to LitharsMending by DigitalZombie: https://github.com/DigitalzombieTLD/LitharsMending

Allows you to restore ruined clothing items. It will automatically chose the worst condition Sewing Kit and then the worst condition Tackle when restoring clothes.

It will consume the default mending materials to restore a ruined item with options to change how difficult it is to repair.

Requires:
- ModSettings
- ComplexLogger
